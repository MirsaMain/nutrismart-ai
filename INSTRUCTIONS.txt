PENYIMPANAN MEAL PLAN AKTIF KE DATABASE

1. Ekstrak ZIP.
2. Salin seluruh isi ke root project nutrismart-ai.
3. Replace file app_pages/meal_plan.py.
4. Jalankan: python -m pytest -q
5. Jalankan: python -m streamlit run streamlit_app.py
6. Buka Meal Plan.
7. Buat plan baru.
8. Klik Simpan sebagai meal plan aktif.
9. Hentikan dan jalankan ulang Streamlit.
10. Pastikan plan aktif tetap tampil.
