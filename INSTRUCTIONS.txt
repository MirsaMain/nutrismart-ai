INTEGRASI RECOMMENDATION ENGINE DAN MEAL PLAN

1. Ekstrak file ZIP.
2. Salin seluruh isinya ke root project nutrismart-ai.
3. Pilih Replace jika Windows menemukan file dengan nama sama.
4. Pastikan sudah ada minimal satu riwayat skrining.
5. Jalankan: python -m pytest -q
6. Jalankan: python -m streamlit run streamlit_app.py
7. Buka menu Meal Plan.
8. Pilih preferensi dan klik Buat meal plan 7 hari.
