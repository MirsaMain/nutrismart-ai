# Alur Sistem

1. Pengguna memasukkan tinggi badan, berat badan, pola makan, dan aktivitas.
2. Sistem menghitung BMI.
3. Pipeline model memproses input pola hidup.
4. Model menghasilkan probabilitas risiko.
5. Probabilitas diubah menjadi skor 0–100.
6. Hasil disimpan berdasarkan tanggal.
7. Riwayat ditampilkan dalam grafik garis.
8. Tanggal tanpa input menggunakan skor terakhir hanya saat grafik dibentuk.
